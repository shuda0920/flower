# 原聲帶區塊使用說明

## 功能概述

已成功添加原聲帶（Soundtrack）區塊到網站中，包含以下功能：

### ✨ 主要功能

1. **旋轉唱片動畫**
   - 精緻的黑膠唱片視覺效果
   - 唱片溝紋細節
   - 金色中心標籤，顯示音樂劇名稱
   - 點擊播放時唱片開始旋轉
   
2. **唱臂動畫**
   - 逼真的唱盤唱臂
   - 播放時唱臂會自動移動到唱片上
   - 停止時唱臂會回到原位

3. **音頻播放器**
   - 播放/暫停按鈕
   - 進度條（可點擊跳轉）
   - 時間顯示（當前時間 / 總時長）
   - 三首試聽曲目：
     - 如果青春會開花 (主題曲) - 4:32
     - 星空下的相遇 - 3:45
     - 夢想的重量 - 4:18

### 📁 新增文件

1. **soundtrack.css** - 原聲帶區塊的所有樣式
2. **soundtrack.js** - 播放器交互功能
3. 已更新 **index.html** - 添加原聲帶區塊和相關引用

### 🎵 如何添加真實音頻文件

當前使用模擬播放功能作為演示。要使用真實音頻文件：

1. 在項目中創建 `audio/` 文件夾
2. 添加音頻文件（推薦 MP3 格式）：
   - `audio/track1.mp3`
   - `audio/track2.mp3`
   - `audio/track3.mp3`

3. 在 **soundtrack.js** 中：
   - 取消註釋底部的真實音頻事件監聽器
   - 註釋掉模擬播放相關代碼

### 🎨 樣式特點

- **金色配色** - 與整體網站主題一致
- **玻璃態效果** - 現代高級感設計
- **響應式設計** - 自動適配手機和平板
- **流暢動畫** - 旋轉和過渡效果

### 📱 響應式支持

- **桌面版** (> 768px): 唱片和播放器並排顯示
- **平板版** (480px - 768px): 適中尺寸，垂直排列
- **手機版** (< 480px): 緊湊佈局，優化觸控操作

### 🔧 自定義曲目

在 **index.html** 中找到 `.track-item` 元素，可以修改：
- 曲目名稱
- 時長顯示
- 添加更多曲目

對應在 **soundtrack.js** 中更新 `durations` 數組。

### 🎭 視覺效果

唱片設計包含：
- 真實的黑膠質感
- 微妙的光澤和反射
- 金色中心標籤
- 旋轉時的唱臂同步動畫
