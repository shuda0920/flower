# 卡司手風琴展示功能說明

## ✨ 功能概述

卡司區塊已改為電影節風格的手風琴展開效果，靈感來自全州國際電影節（JIFF）的視覺設計。

## 🎨 視覺效果

### 默認狀態（窄條）
- 6個演員卡片並排顯示
- 每個卡片寬度相等，呈現窄條狀
- 顯示橘紅色分類標籤（如"開場影片"、"韓國競賽"等）
- 演員姓名垂直顯示（書寫模式：直排）
- 背景為深灰漸變
- 卡片之間有金色分隔線

### 懸停/展開狀態
- 被懸停的卡片展開至4倍寬度
- 顯示完整演員肖像（左側 60%）
- 顯示詳細信息（右側 40%）：
  - 分類標籤（右上角）
  - 演員姓名（大字體）
  - 角色名稱（金色）
  - 角色描述（灰色文字）
- 其他卡片縮小並變暗（50%亮度）
- 平滑動畫過渡（0.8秒）

## 🎯 交互功能

### 桌面端
- **鼠標懸停** - 展開卡片顯示完整信息
- **點擊** - 固定展開狀態（添加 active class）
- **再次點擊** - 保持展開（不會收縮）
- **鍵盤方向鍵** - ←→ 切換上/下一位演員

### 移動端（< 768px）
- **垂直排列** - 卡片改為上下堆疊
- **點擊** - 展開卡片
- **觸控滑動** - 上下滑動切換演員
- **自動調整** - 分類標籤和姓名改為橫向顯示

### 觸控設備
- **滑動手勢** - 支持左右/上下滑動切換
- **最小距離** - 50px 觸發滑動
- **方向判斷** - 根據屏幕方向自動選擇橫向或縱向滑動

## 📐 響應式設計

### 桌面 (> 1024px)
- 卡片高度：600px
- 展開比例：1:4
- 姓名字體：3.5rem

### 平板 (769-1024px)
- 卡片高度：500px
- 姓名字體：2.8rem
- 描述字體：0.9rem

### 手機 (≤ 768px)
- **垂直佈局**
- 默認高度：100px
- 展開高度：500px
- 姓名字體：2.2rem
- 分類標籤和姓名改為水平排列

### 小屏手機 (≤ 480px)
- 默認高度：80px
- 展開高度：450px
- 姓名字體：1.8rem

## 🎨 設計細節

### 顏色方案
- **分類標籤背景**：#FF5722（橘紅色，致敬JIFF）
- **姓名顏色**：#FFFFFF（白色）
- **角色顏色**：#D4AF37（金色）
- **描述顏色**：#B0B0B0（灰色）
- **卡片背景**：漸變灰（#1a1a1a → #0a0a0a）

### 動畫時序
- **展開/收縮**：0.8s cubic-bezier(0.4, 0, 0.2, 1)
- **內容淡入**：0.6s ease + 0.2s延遲
- **窄條淡出**：0.4s ease
- **亮度變化**：與展開同步

### 分隔線
- 位置：每個卡片右側
- 顏色：金色漸變（#D4AF37, 30%透明度）
- 高度：90%（上下各留5%）
- 漸變：頂部和底部淡出

## 💡 使用技巧

### 1. 更改默認展開的卡片
在HTML中將 `active` class 移到想要的卡片：
```html
<div class="accordion-card active" data-cast="3">
```

### 2. 添加真實圖片
替換placeholder：
```html
<div class="expanded-image">
    <img src="images/cast-member-3.jpg" alt="陳雨柔">
</div>
```

### 3. 自定義分類標籤顏色
在CSS中修改：
```css
.cast-label {
    background: #你的顏色;
}
```

### 4. 調整展開比例
在CSS中修改flex值：
```css
.accordion-card:hover,
.accordion-card.active {
    flex: 4;  /* 改為其他數字，如5或3 */
}
```
