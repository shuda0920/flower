# 主創團隊合照功能更新說明

## ✨ 新增功能

主創團隊輪播現在以**團隊大合照**作為第一頁（第0頁），並實現了自動重置功能。

### 📸 團隊合照設計

#### 視覺效果
- **置中顯示**：照片在頁面中央位置
- **灰色占位塊**：700x500px 的漸層灰色區塊
- **微光動畫**：背景有緩慢的漸層移動效果
- **金色光暈**：輕微的金色光暈點綴

#### 🖱️ 鼠標懸停效果

當鼠標懸停在照片上時：

```
┌─────────────────────────────────┐
│ 遮罩區域    │                   │
│ (黑色漸層)  │   照片原始顯示    │
│             │                   │
│ 如果青春會開花 │                │
│ 主創團隊       │                │
│ 描述文字...    │                │
└─────────────────────────────────┘
```

- **左半遮罩**：黑色漸層遮罩從左側覆蓋50%
- **右半透明**：右側50%保持照片原樣
- **介紹詞顯示**：
  - 劇名「如果青春會開花」（金色）
  - 副標題「主創團隊」（金色，大寫）
  - 簡短介紹文字（灰色）
- **滑入動畫**：內容從左側滑入，流暢過渡

### 🔄 自動重置功能

#### 触發條件

當用戶離開主創團隊區塊後再回來時：
1. ✅ **向上滾動**回到團隊區塊 → 自動重置到合照頁
2. ✅ **向下滾動**回到團隊區塊 → 自動重置到合照頁
3. ✅ **點擊導航**跳轉到團隊區塊 → 自動重置到合照頁

#### 實現原理

使用 `IntersectionObserver` API：
- 監測團隊區塊的可見性
- 閾值設置為 20%（區塊顯示20%時觸發）
- 記錄用戶是否離開過區塊
- 回到區塊時自動切換到第0頁
