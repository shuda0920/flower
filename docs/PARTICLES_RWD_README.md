# 粒子特效響應式設計說明

## 🎨 更新內容

p5.js 粒子背景效果現已完全支持響應式設計（RWD），會根據屏幕尺寸自動調整。

## 📐 響應式參數配置

### 粒子數量調整

根據屏幕寬度動態調整粒子數量，優化性能：

| 屏幕尺寸 | 寬度範圍 | 粒子數量 | 玫瑰縮放 |
|---------|---------|---------|---------|
| 📱 手機 | < 480px | 300 | 0.5x |
| 📱 平板 | 480-768px | 500 | 0.7x |
| 💻 小桌面 | 768-1200px | 650 | 0.85x |
| 🖥️ 大桌面 | > 1200px | 800 | 1.0x |

### 粒子大小調整

- **桌面版** (≥ 768px): 2-4px
- **移動版** (< 768px): 1.5-3px

### 玫瑰形狀縮放

玫瑰圖案會根據屏幕尺寸縮放：

- **手機**: 50% 大小
- **平板**: 70% 大小
- **小桌面**: 85% 大小
- **大桌面**: 100% 原始大小

## 🔧 實現原理

### 1. 初始化檢測

```javascript
function updateResponsiveParams() {
    let w = window.innerWidth;
    
    if (w < 480) {
        numParticles = 300;
        roseScale = 0.5;
    } else if (w < 768) {
        numParticles = 500;
        roseScale = 0.7;
    }
    // ...
}
```

### 2. 窗口調整處理

當用戶調整瀏覽器窗口大小時：

1. ✅ 重新計算畫布尺寸
2. ✅ 更新響應式參數
3. ✅ 重新初始化粒子（如果數量變化超過50個）

### 3. 性能優化

- **條件重初始化**: 只有當粒子數量變化超過50個時才重新生成粒子
- **減少移動端粒子**: 移動設備使用更少的粒子，提升流暢度
- **動態大小**: 粒子大小根據屏幕自動調整

### 窗口調整邏輯

```javascript
p.windowResized = () => {
    p.resizeCanvas(window.innerWidth, window.innerHeight);
    
    let oldNumParticles = numParticles;
    updateResponsiveParams();
    
    // 只有變化超過50個才重新初始化
    if (Math.abs(oldNumParticles - numParticles) > 50) {
        particles = [];
        for (let i = 0; i < numParticles; i++) {
            particles.push(new Particle(p, i, numParticles));
        }
    }
};
```
