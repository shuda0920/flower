# 主創團隊輪播功能說明

## ✨ 更新內容

主創團隊區塊已重新設計為**5人輪播展示**，採用左右布局，一次顯示一位成員的詳細介紹。

### 👥 團隊成員（5位）

1. **林星辰** - 導演 (Director)
2. **張微光** - 編劇 (Playwright)
3. **陳律動** - 音樂總監 (Music Director)
4. **吳流雲** - 編舞 (Choreographer) ⭐ 新增
5. **黃築夢** - 舞台設計 (Set Designer) ⭐ 新增

### 📐 布局設計

採用**左右分欄布局**：

```
┌─────────────────────────────────────────┐
│  [照片]          職位徽章               │
│  350x450         姓名                   │
│  占位圖          個人語錄               │
│                  詳細經歷介紹           │
│                  代表作品               │
└─────────────────────────────────────────┘
```

### 🎮 互動功能

1. **左右箭頭導航**
   - 點擊左箭頭：查看上一位成員
   - 點擊右箭頭：查看下一位成員
   - 循環瀏覽（最後一位 → 第一位）

2. **底部指示器**
   - 5個圓點指示器
   - 當前成員的指示器會變長並高亮
   - 可點擊指示器直接跳轉到對應成員

3. **鍵盤導航**
   - ⬅️ 左方向鍵：上一位
   - ➡️ 右方向鍵：下一位
   - 僅在團隊區塊可見時啟用

4. **觸摸滑動（手機）**
   - 向左滑：下一位
   - 向右滑：上一位

### 🎨 設計特色

- **金色配色**：職位徽章、姓名使用金色主題
- **玻璃效果**：導航按鈕採用毛玻璃效果
- **流暢動畫**：切換時有淡入滑動效果
- **響應式設計**：自動適配手機和平板

### 📱 響應式布局

#### 桌面版 (> 968px)
- 左右並排布局
- 照片固定 350x450px
- 完整顯示所有介紹內容

#### 平板版 (640px - 968px)
- 改為上下布局
- 照片居中，最大 300x380px
- 文字內容居中對齊

#### 手機版 (< 640px)
- 垂直堆疊布局
- 照片縮小至 250x320px
- 文字大小調整
- 導航箭頭移至邊緣

### 📂 新增文件

1. **team-carousel.css** - 團隊輪播樣式
   - 輪播容器和幻燈片
   - 照片和資訊區塊樣式
   - 導航箭頭和指示器
   - 響應式斷點

2. **team-carousel.js** - 輪播互動功能
   - 成員切換邏輯
   - 箭頭和指示器事件
   - 鍵盤導航
   - 觸摸滑動支持

### 🖼️ 添加真實照片

當準備好團隊照片時：

1. 將照片放置在 `images/team/` 資料夾
2. 在 `index.html` 中替換占位符：

```html
<div class="team-photo">
    <img src="images/team/director.jpg" alt="林星辰">
</div>
```

### 🎭 成員介紹結構

每位成員包含：
- **職位徽章**：中英文職位
- **姓名**：大標題顯示
- **個人語錄**：斜體金色文字
- **詳細經歷**：2-3段文字介紹
- **代表作品**：列舉重要作品
